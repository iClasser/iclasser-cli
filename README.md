# iclasser-cli

iClasser command line tool

- [Installation](#installation)
- [Usage](#usage)
- [Commands](#commands)

# Installation

```sh
npm install -g iclasser-cli
# or
yarn global add iclasser-cli
```


# Usage

```sh
iclasser COMMAND
# runs the command
iclasser (-v|--version|version)
# prints the version
iclasser --help COMMAND
# outputs help for specific command
```
